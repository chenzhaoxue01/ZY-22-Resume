Jangouts - 视频会议系统

基于 WebRTC 和 Janus Gateway 的自托管视频会议解决方案

项目概述

Jangouts (Janus Hangouts) 是一个完全自托管的开源视频会议替代方案，界面设计灵感来自 Google Hangouts。

主要功能：

  • 视频会议
  • 音频通话
  • 屏幕共享
  • 文字聊天
  • 无限会议室
  • 可配置参与者限制

运行要求

必需组件：

  1. Node.js 和 npm - 用于构建和运行前端应用
  2. Janus Gateway - WebRTC 媒体服务器
  3. Web 服务器 - 用于提供静态文件

当前状态：项目代码已成功克隆到本地，但需要安装Node.js才能正常运行。

下一步操作

选项1：安装Node.js（推荐）

  1. 访问 nodejs.org 下载并安装Node.js
  2. 打开命令提示符，导航到项目目录
  3. 运行 npm install 安装依赖
  4. 运行 npm start 启动开发服务器

选项2：使用Docker

  1. 安装Docker Desktop
  2. 构建Janus Gateway容器
  3. 配置Web服务器提供静态文件

项目文件结构：

  • src/ - React源代码
  • public/ - 静态资源文件
  • package.json - 项目配置和依赖
  • README.md - 详细文档
